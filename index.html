Bootstrap Table JS

Легко настраиваемая JS таблица для Bootstrap 5 или более поздних версий

Базовые возможности и преимущества:

  • Не большой вес (менее 10 Kb);
  • Автоматическое построение таблицы на основе массива объектов;
  • Фильтрация данных;
  • Сортировка данных;
  • Постепенная отрисовка данных, по мере скрола таблицы (lazy loading);
  • Закрепление строки заголовков;
  • Возможность замены строк данных по шаблону, указанному в настройках;
  • Гибкая настройка действий с элементами данных

Пример:

Быстрый старт

Создайте таблицу table в файле .html и пропишите идентификатор id="btable"

Подключите скрипт: <script src="js/bootstrapTable.js"></script>

Создайте таблицу данных в файле .jsconst btable = new BsTable('btable', data, options);

  • 'btable' - Id таблицы
  • data - массив объектов
  • options - настройки таблицы (не обязательный параметр)

Настройки:

  • itemsToPage - количество записей для начального показа. Количество записей для отображения при скроле таблицы
  • hiddenColumns - какие поля объекта следует скрыть. Пример: ["phone", "name", "id"]
  • tableHeight - максимальная высота таблицы данных. Пример: "500px"
  • fixedHeaderTable - следует ли фиксировать заголовок таблицы при скроле. Пример: true
  • renameColumn - массив объектов для переименования названия полей. Пример: renameColumn: [ { key: "id", replace: "ИД" }, { key: "name", replace: "Имя" } ],
  • replace - заменить значение полей. Массив объектов.

    Пример 1:

    { value: "true", key: "verify", replace: `<span class="badge text-bg-success">да</span>` },

    В данном примере будет заменено значение true в поле verify на <span class="badge text-bg-success">да</span>

    Пример 2:

    { value: "{value}", key: "name", replace: `<a href="{value}" target="_blank">{value}</a>` },

    В данном примере будет заменено любое значение поля name на <a href="{value}" target="_blank">{value}</a>. Где {value} - любое значение поля

  • actions - массив строк. Для элемента управления записью (отображается при клике по строке таблицы) Пример: `<a class="btn btn-sm btn-danger" href="javascript:remove('{name}')"><i class="bi bi-trash3"></i></a>`
